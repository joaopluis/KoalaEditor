An awesome jQuery WYSIWYG HTML text editor.

Easy to integrate and with a simple flat design. Also, with a cute koala as a mascot.

Examples

1. <textarea>

$("#editable").koala();

2. <div>

Some dummy content in this div.

$("#editable2").koala();

3. With image upload

Some dummy content in this div.

$("#editable3").koala({
    uploadURL: "upload.php"
});

4. No toolbar

Some dummy content in this div.

$("#editable4").koala({
    toolbar: false
});

5. Only some buttons

Some dummy content in this div.

$("#editable5").koala({
    buttons: ['bold', 'italic', 'underline', 'sep', 'code']
});

6. The dark side

Some dummy content in this div.

$("#editable6").koala({
    theme: "dark-koala"
});

7. Now in Portuguese!

Algum conteúdo aleatório nesta div.

$("#editable7").koala({
    language: "pt_pt"
});
